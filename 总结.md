## 二、功能实现

### 2.1 小队管理 API 补全（6 个新增接口）

| 序号 | 接口 | 说明 |
|:---:|------|------|
| 1 | `POST /teams/{id}/transfer-leader` | 转让队长，原队长降级为管理员 |
| 2 | `POST /teams/{id}/blacklist` | 将用户加入黑名单（自动移出成员） |
| 3 | `DELETE /teams/{id}/blacklist/{userId}` | 移出黑名单 |
| 4 | `GET /users/me/teams` | 查看我加入的所有小队及角色 |

此外还为 Admin 小队列表增加了 `q`（名称搜索）和 `status`（状态筛选）参数。

### 2.2 首页小队推荐流

- `GET /discover/teams?tab=recommended|latest` — 推荐 Tab 按成员数降序，最新 Tab 按创建时间降序，支持游标分页
- 实现于 `DiscoveryService` / `DiscoveryServiceImpl` / `DiscoveryController`

### 2.3 小队群聊

- `GET /teams/{id}/chat` — 返回群聊信息（最新消息、发送者昵称、未读数）
- 创建小队 / 成员加入 / 审批通过时自动初始化 `group_chat_read_markers`
- 群聊实时通讯通过已有 IM WebSocket（`entity_type=group`, `entity_id=team:{teamId}`）
