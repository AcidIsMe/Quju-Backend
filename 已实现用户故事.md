### 小队模块 API 详情（迭代2 完善）

> 更新时间：2026-07-04  
> 所有小队 API 均需 JWT 认证

#### 用户端 API (`/api/teams`)

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| POST | `/teams` | 创建小队 | 登录用户 |
| GET | `/teams` | 小队发现列表（支持 `?q=` 搜索） | 登录用户 |
| GET | `/teams/{id}` | 小队详情 | 登录用户 |
| PUT | `/teams/{id}` | 修改小队信息 | 队长/管理员 |
| POST | `/teams/{id}/join` | 加入小队 | 登录用户 |
| POST | `/teams/{id}/leave` | 退出小队 | 成员 |
| POST | `/teams/{id}/dissolve` | 解散小队 | 队长 |
| PATCH | `/teams/{id}/members/{userId}/role` | 修改成员角色 | 队长 |
| DELETE | `/teams/{id}/members/{userId}` | 移除成员 | 队长/管理员 |
| GET | `/teams/{id}/members` | 成员列表 | 登录用户 |
| GET | `/teams/{id}/join-requests` | 入队申请列表 | 队长/管理员 |
| POST | `/teams/{id}/join-requests/{requestId}/approve` | 通过入队申请 | 队长/管理员 |
| POST | `/teams/{id}/join-requests/{requestId}/reject` | 拒绝入队申请 | 队长/管理员 |
| GET | `/teams/{id}/leaderboard` | 积分排行榜 | 登录用户 |
| POST | `/teams/{id}/points` | 增减积分 | 队长/管理员 |
| POST | `/teams/{id}/transfer-leader` | **【新增】转让队长** | 队长 |
| POST | `/teams/{id}/blacklist` | **【新增】加入黑名单** | 队长/管理员 |
| DELETE | `/teams/{id}/blacklist/{userId}` | **【新增】移出黑名单** | 队长/管理员 |
| GET | `/users/me/teams` | **【新增】我的小队列表** | 登录用户 |

#### 管理端 API (`/api/admin`)

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| GET | `/admin/teams` | 小队管理列表（支持 `?q=&status=`） | admin |
| GET | `/admin/teams/{id}` | 小队详情（含成员列表） | admin |
| GET | `/admin/teams/{id}/members` | 小队成员列表 | admin |
| POST | `/admin/teams/{id}/disable` | 停用小队（需填写原因） | admin |
| POST | `/admin/teams/{id}/restore` | 恢复小队 | admin |

#### 新增功能说明

1. **转让队长** (`POST /teams/{id}/transfer-leader`)：队长可将队长身份转让给小队内任意成员，原队长自动降级为管理员
2. **黑名单管理**：队长/管理员可将用户加入黑名单（自动移出成员），也可解除黑名单
3. **我的小队** (`GET /users/me/teams`)：查看当前用户加入的所有小队及角色
4. **重复申请检查**：`join()` 方法新增已提交 pending 申请的重复检测
